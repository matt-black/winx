# `lax`

A `jax.lax`-like API that allows filters to be applied similar to how convolutions are applied in `jax`.

::: winx.lax.filt
    handler: python
    options:
        show_source: false
        show_root_heading: true

::: winx.lax.filt_general_dilated
    handler: python
    options:
        show_source: false
        show_root_heading: true
